# 💊 MediSebi – Smart Pharmacy Operating System

🚀 Live Demo: https://medisebi.lovable.app/

---

## 🧠 Overview

**MediSebi** is an AI-powered pharmacy management system designed to help independent pharmacies operate smarter, reduce losses, and improve customer satisfaction.

Built using modern web technologies and AI-driven workflows, MediSebi transforms traditional pharmacy operations into a **data-driven, intelligent system**.


---

## 🎯 Problem

Independent pharmacies face major challenges:

* ❌ Manual billing & inventory tracking
* ❌ No demand forecasting
* ❌ Medicine expiry losses (~5%)
* ❌ Stock-outs leading to lost customers (~15%)

👉 Result: **Revenue loss + customer dissatisfaction**

---

## 💡 Solution

MediSebi provides a **complete pharmacy OS** with:

* ✅ Smart billing system
* ✅ Real-time inventory tracking
* ✅ AI-based demand prediction
* ✅ Automated restocking
* ✅ Expiry & low-stock alerts
* ✅ Local pharmacy trading network

---

## ⚙️ Features

### 💳 Smart Billing

* Add/search medicines instantly
* Auto GST calculation (CGST/SGST/IGST)
* Discount system (flat + percentage)
* Real-time bill summary
* Multiple payment methods (UPI, Card, Cash, Net Banking)

---

### 📦 Inventory Management

* Live stock tracking
* Expiry alerts
* Low-stock notifications
* Auto inventory updates after each sale

---

### 🧠 AI Demand Prediction

* Uses time-series ML models
* Predicts future demand
* Reduces overstocking & stock-outs

---

### 🔄 Automated Restocking

* Smart restock suggestions
* Maintains optimal inventory levels
* Eliminates manual guesswork

---

### 🌐 Local Pharmacy Network (USP)

* Connect nearby pharmacies
* Trade surplus medicines
* Reduce expiry losses
* Convert dead stock → revenue

---

## 📊 Impact

* 💰 Recover up to **5% revenue lost due to expiry**
* 📈 Improve **customer retention by 15%**
* ⚡ Faster billing & operations
* 🤝 Build collaborative pharmacy ecosystem

## 🚀 Getting Started

### 1️⃣ Clone the repo

```bash
git clone https://github.com/your-username/medisebi.git
cd medisebi
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create `.env` file:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_KEY=your_key
```

### 4️⃣ Run the project

```bash
npm run dev
```

---

## 🔐 Environment Variables

| Variable          | Description          |
| ----------------- | -------------------- |
| VITE_SUPABASE_URL | Supabase project URL |
| VITE_SUPABASE_KEY | Supabase API key     |
